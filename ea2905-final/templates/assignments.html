Summaries of the assignments

  • In homework 1 we converted binary numbers from positive to one's complement, two's complement, and more. We also did some list manipulation for the programming part.
  • In homework 2 we did work with sorting and searching algorithms such as selection sort and binary search. We did some projectile simulations for the programming part.
  • In homework 3 we explored recursion and implemented some functions using it. For the programming part we simulated water percolation.
  • In homework 4 we created some objects and plotted statistics, such as "class average" and "Teacher pay".
  • In homework 5 we used the pandas library to calculate and plot things like "skewness" and "kurtosis".
